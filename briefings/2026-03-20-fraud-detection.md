# Fraud Detection Research Briefing

## Papers/Research

1. **A Multi-Scale Graph Learning Framework with Temporal Consistency Constraints for Financial Fraud Detection in Transaction Networks under Non-Stationary Conditions** (Relevance: 10)
   Addresses modeling sparse anomalies, dynamic patterns, and severe class imbalance with temporal drift in financial transaction networks. Authors: Yiming Lei, Qiannan Shen, Junhao Song | Mar 2026

2. **Privacy-Preserving Federated Fraud Detection in Payment Transactions with NVIDIA FLARE** (Relevance: 10)
   Explores federated learning as a solution for collaborative fraud detection across institutions while addressing regulatory and data-sovereignty constraints. Authors: Holger R. Roth, Sarthak Tickoo, Mayank Kumar | Mar 2026

3. **EmDT: Embedding Diffusion Transformer for Tabular Data Generation in Fraud Detection** (Relevance: 10)
   Proposes a clustered embedding diffusion transformer to generate synthetic tabular data, mitigating class imbalance issues in fraud datasets. Authors: En-Ya Kuo, Sebastien Motsch | Mar 2026

4. **A Depth-Aware Comparative Study of Euclidean and Hyperbolic Graph Neural Networks on Bitcoin Transaction Systems** (Relevance: 8)
   Compares Euclidean vs hyperbolic GNNs for entity detection and transaction classification in Bitcoin networks using multi-hop interaction patterns. Authors: Ankit Ghimire, Saydul Akbar Murad, Nick Rahimi | Mar 2026
