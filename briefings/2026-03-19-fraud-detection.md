# Fraud Detection Research Briefing

## Papers/Research

- **A Multi-Scale Graph Learning Framework with Temporal Consistency Constraints for Financial Fraud Detection in Transaction Networks under Non-Stationary Conditions** (Relevance: 10/10) — Addresses modeling sparse anomalies, dynamic patterns, and severe class imbalance with temporal drift in financial transaction networks. Authors: Yiming Lei, Qiannan Shen, Junhao Song. Published 2026-03-15.

- **Privacy-Preserving Federated Fraud Detection in Payment Transactions with NVIDIA FLARE** (Relevance: 10/10) — Explores Federated Learning as a solution for collaborative fraud detection across institutions while addressing regulatory, privacy, and data-sovereignty constraints. Authors: Holger R. Roth, Sarthak Tickoo, Mayank Kumar. Published 2026-03-13.

- **EmDT: Embedding Diffusion Transformer for Tabular Data Generation in Fraud Detection** (Relevance: 10/10) — Proposes a Clustered Embedding Diffusion Transformer to generate synthetic fraud data, addressing class imbalance issues where classifiers are biased toward legitimate transactions. Authors: En-Ya Kuo, Sebastien Motsch. Published 2026-03-13.

- **A Depth-Aware Comparative Study of Euclidean and Hyperbolic Graph Neural Networks on Bitcoin Transaction Systems** (Relevance: 7/10) — Compares Euclidean vs. hyperbolic GNNs for entity detection and transaction classification in Bitcoin networks using multi-hop interaction patterns. Authors: Ankit Ghimire, Saydul Akbar Murad, Nick Rahimi. Published 2026-03-17.
