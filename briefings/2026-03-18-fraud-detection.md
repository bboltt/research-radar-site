# Fraud Detection Research Briefing

## Papers/Research

### Multi-Scale Graph Learning for Financial Fraud Detection
- **Authors:** Yiming Lei, Qiannan Shen, Junhao Song
- **Published:** March 15, 2026
- **Relevance:** 10/10
- **Summary:** Addresses modeling sparse anomalies, dynamic patterns, and severe class imbalance in transaction networks under non-stationary conditions. Proposes temporal consistency constraints for financial fraud detection.
- **URL:** http://arxiv.org/abs/2603.14592v1

### Privacy-Preserving Federated Fraud Detection with NVIDIA FLARE
- **Authors:** Holger R. Roth, Sarthak Tickoo, Mayank Kumar
- **Published:** March 13, 2026
- **Relevance:** 10/10
- **Summary:** Explores federated learning for collaborative fraud detection across institutions while preserving privacy and addressing data sovereignty constraints.
- **URL:** http://arxiv.org/abs/2603.13617v1

### EmDT: Embedding Diffusion Transformer for Tabular Data Generation
- **Authors:** En-Ya Kuo, Sebastien Motsch
- **Published:** March 13, 2026
- **Relevance:** 10/10
- **Summary:** Proposes synthetic data generation using diffusion transformers to address class imbalance in fraud detection datasets.
- **URL:** http://arxiv.org/abs/2603.13566v1
