# Fraud Detection Research Briefing

## Papers/Research

- **A Multi-Scale Graph Learning Framework with Temporal Consistency Constraints for Financial Fraud Detection in Transaction Networks under Non-Stationary Conditions** (Relevance: 10) — Addresses modeling sparse anomalies, dynamic patterns, and severe class imbalance with temporal drift in financial transaction networks. [arxiv](http://arxiv.org/abs/2603.14592v1)

- **Privacy-Preserving Federated Fraud Detection in Payment Transactions with NVIDIA FLARE** (Relevance: 10) — Explores Federated Learning to enable collaborative fraud detection across institutions while respecting privacy and data-sovereignty constraints. [arxiv](http://arxiv.org/abs/2603.13617v1)

- **EmDT: Embedding Diffusion Transformer for Tabular Data Generation in Fraud Detection** (Relevance: 10) — Proposes a clustered embedding diffusion transformer to generate synthetic tabular data for handling imbalanced fraud datasets. [arxiv](http://arxiv.org/abs/2603.13566v1)

- **A Depth-Aware Comparative Study of Euclidean and Hyperbolic Graph Neural Networks on Bitcoin Transaction Systems** (Relevance: 8) — Compares Euclidean vs hyperbolic GNNs for entity detection and transaction classification in Bitcoin networks. [arxiv](http://arxiv.org/abs/2603.16080v1)
